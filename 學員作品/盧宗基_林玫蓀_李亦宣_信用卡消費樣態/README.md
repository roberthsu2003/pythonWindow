## 專題目的：
依性別、職業、年齡層、教育程度及年收入等多方面因素進行分析，透過資料視覺化呈現消費者使用信用卡的消費習慣性與趨勢。

1. 資料來源：聯合信用卡處理中心Open API
2. 資料範圍：2014年1月－2023年8月
3. 程式：main.py、data.py
